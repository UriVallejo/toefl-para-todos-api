PARA CREAR UN CONTROLADOR, 
1-.SE CREA EL CONTROLADOR DE LA ENTIDAD NUEVA
    -EN EL CONTROLADOR SE IMPORTAN LAS DEPENDENCIAS O LIBRERIAS
    -SE IMPORTAN LOS MODELOS QUE UTILIZARA EL CONTROLADOR,
    -SE CREAN LAS FUNCIONES DEL CONTROLADOS
    -SE EXPORTA EL CONTROLADOR EN LA PARTE INFERIOR DEL DOCUMENTO.
2-. SE CREA EL FICHERO DE RUTAS
    -SE IMPORTAN LAS DEPENDENCIAS NECESARIAS COMO EXPRESS,
    -SE IMPORTA EL CONTROLADOR CREADO EN EL PASO ANTERIOR,
    -SE CREA EL API CON EXPRESS ROUTER
    -sE IMPORTAN LOS MIDDLEWARES
    -SE INDICA DONDE SE GUARDARAN LOS ARCHIVOS EN CASO DE UTILIZAR MULTIPART
    -SE CREA LA CARPETA QUE SE MANDO COMO REFERENCIA EN EL PASO ANTERIOR
    -SE CREAN LAS RUTAS 
    -SE EXPORTA EL MODULO IGUALANDO A API ( EXPRESS); 

3-.  CARGAR TODO LO CREADO ANTERIORMENTE EN APP.JS
    -CARGAR EN UNA VARIABLE EL FICHERO DE RUTAS DE LA ENTIDAD NUEVA
    CREAR EL API.USE QUE ES CORRESPONDIENTE A LA NUEVA ENTIDAD
